## ✨ Visão Geral

O **Fintech Dashboard** foi criado para reproduzir uma interface financeira com visual **dark**, layout sofisticado e foco em **clareza visual**, **organização dos dados** e **boa experiência em diferentes tamanhos de tela**.

Este projeto foi desenvolvido com o objetivo de entregar uma interface:

- moderna
- responsiva
- limpa
- visualmente forte
- fácil de manter
- pronta para evoluir para aplicações reais

---

## 🚀 Preview

### Desktop
Interface inspirada em dashboards financeiros modernos, com cards informativos, navegação superior e área de saldo destacada.

### Mobile
Layout adaptado para telas menores, reorganizando os blocos para manter legibilidade e usabilidade.

---

## 🧠 Conceito do Projeto

A proposta visual deste dashboard gira em torno de três pilares:

- **Leitura rápida de informações**
- **Hierarquia visual forte**
- **Estética premium em dark mode**

Os elementos principais da interface incluem:

- Navbar superior com destaque para a página ativa
- Cards de receita, despesas e investimentos
- Card de último gasto
- Card principal de saldo com gráfico circular
- Layout fluido para desktop, tablet e mobile
